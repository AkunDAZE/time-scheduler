# 作業記録アプリ説明書 <日本語版>

## DL方法
**1**. [リリースページ](https://github.com/AkunDAZE/time-scheduler/releases)の一番上にある **Latest** を探してください。　

![参考画像](./Sample1.png)
  

**2**. *Assets* の「**Setup.exe**」の方をクリックしてダウンロードしてください。  
`※ Setup.exe.blockmap ではないので注意`  
___
  ## 使い方
  ### 記録を開始して記録するまでの方法
  <details>
  <summary>ここをクリックして表示</summary>
    
    1：画面中央右にある「作業内容を追加」を押す。
    
    2：左の丸を押してお好みの色を選択する。
    
    3：作業の名前とお好みでメモを追加して作成を押す。
    
    4：作成したタスクの再生ボタンを押して記録開始する。

    5：「停止して保存」を押して記録を保存する。
  </details>

___
  
  ### 保存した記録を確認する方法
  <details>
  <summary>ここをクリックして表示</summary>
    
    1：画面右上にある「分析情報」を押す。

    2：「作業内訳」と「グラフ」に記録したタスクがあるか確認する。
  </details>
